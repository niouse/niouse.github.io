Simple fonction ou composant react ?

Pour créer un composant react, une simple fonction suffit :

const MyComponent = function(){ return (
My super component
) }

Et oui, on peut renvoyer directement du "html", enfin quelquechose qui ressemble à du html et ensuite réutiliser MyComponent comme une balise html :

Pour que cela fonctionne, il y a un prérequis : il faut que l'objet "react" (contenant une partie de l'api react) soit dans le scope de la fonction. La raison est très simple, la notation "" est une autre représentation du javascript. Cela ressemble à du html mais cela représente une instruction en javascript, en effet, les deux lignes suivante sont parfaitement interchangeables dans du jsx:

MyComponent

et

React.createElement(MyComponent) Il suffit donc d'importer react au début du fichier : import react from "react"
